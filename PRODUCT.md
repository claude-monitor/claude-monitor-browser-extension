# Product

## Register

product

## Users

Claude Pro / Max / Team subscribers, especially Claude Code power users on Max plans and teams using Claude Design. They keep the extension pinned and glance at the toolbar mid-task to answer one question: how close am I to a limit, and when does it reset? Context is a working session inside claude.ai, an IDE, or the browser: a quick glance, then back to work.

## Product Purpose

A toolbar popup that reads Claude.ai usage straight from the authenticated session, so the user never opens claude.ai/settings/usage. It shows the 5-hour session, the 7-day weekly limit (all models), per-model weekly sub-caps (Opus, Sonnet, Claude Design), the daily routine-run count, the plan badge, and extra credits. Values are color-coded against thresholds so the answer is pre-attentive. Success: the user never gets surprised by a limit.

## Brand Personality

An instrument, not a dashboard with opinions. Honest, precise, calm, and warm rather than clinical. Because it tracks Claude, it speaks the real Anthropic visual language (clay on warm near-black) instead of the generic "AI equals purple" reflex. Three words: trustworthy, warm, exact.

## Anti-references

- Generic "AI tool" dark plus neon purple/violet. That was the prior look and the category reflex; purple was never Anthropic's color.
- SaaS hero-metric marketing styling: giant gradient numbers, decorative gradients, glow.
- Crypto / trading-terminal neon-on-black density.
- Over-decorated controls or gratuitous motion that makes a glanceable utility feel busy.

## Design Principles

- **Pre-attentive first.** State (color, number, bar) reads in a glance, before focusing. Everything else is support.
- **Speak Claude's real language.** Warm neutrals and clay, the actual Anthropic palette: honest about what it monitors, not a purple stand-in.
- **The tool disappears.** Earned familiarity over novelty. Standard affordances, consistent vocabulary, no surprises.
- **Color carries meaning.** Green / amber / red is data (usage level). The clay accent is for action and selection only, never decoration.
- **Restraint scales.** Optional cards and dense info only when the plan exposes them. Quiet by default.

## Accessibility & Inclusion

- WCAG AA contrast on all text (>=4.5:1 body, >=3:1 large) against the warm-dark surfaces.
- Color is never the only signal: every color-coded value is also a number and a bar width.
- Honor `prefers-reduced-motion` (no spinner or bar animation when set).
- Visible keyboard focus on every interactive control.
